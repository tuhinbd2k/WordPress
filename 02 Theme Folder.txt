02 Theme Folder
=========================


2.1 Theme Folder Creation:
We've to create a theme folder with the name of our new theme like "mytheme" at the followin 
Location:

Root-Directory > wp-content\themes\


2.2 Theme File Creation:
We've to create three files uder "mytheme" to recognize and activate the "mytheme" folder as a theme 
by WordPress, those are as follows:

a. Index File  -> index.php
b. Style Sheet -> style.css
c. Thumbnail   -> screenshort.png

File name should same as stated above.


2.3 The style.css File:
We've to write the following codes as comments in the "style.css" file for a valid CSS to make the 
theme:

/*
Theme Name: 
Theme URI: 
Author: 
Author URI: 
Version: 
Description: 
*/


2.4 The screenshot.png file:
We've to make a "screenshot.png" to view a thumbnail of the theme. The dimension should be 600px X 450px 
(width X height). We can it in jpg format also.
